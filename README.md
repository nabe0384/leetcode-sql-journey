## 🔧 Operating Rules

### 継続のためのルール

- 連続記録より、週単位の継続を重視
- 疲れた日は5分タスクでOK(問題読むだけ、解説読むだけ)
- 1週間休んでも復帰すれば継続扱い
- メンタル3以下の日は休む
- やらない日の罪悪感を捨てる

### 優先順位(体力・時間が足りない時)

1. 睡眠・健康
2. 筋トレ(週2-3回、習慣維持)
3. SQL(平日月火木)
4. 時系列本(水金)
5. 週末学習(評価指標・ドメイン)
6. 簿記(通勤時間メイン、試験前は優先UP)
7. デジマ本(余裕があれば)

### 学習配分(週)

- 月: SQL Easy 1-2問(40分)
- 火: SQL Easy 1-2問(40分)
- 水: 時系列・状態空間モデル本(40分)
- 木: SQL Easy 1-2問(40分)
- 金: 異常検知本(40分)
- 土: 3時間集中(評価指標・実装・Qiita)
- 日: 2時間(デジマ本・振り返り)
- 通勤: 簿記(6/14まで)、以降はSQL復習
- 筋トレ: 週2-3回
